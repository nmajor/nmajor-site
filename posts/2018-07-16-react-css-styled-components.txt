b:T13b1,<p>Styled Components allow us to bundle the templating (HTML), behavior (JS), and styling (CSS) all together. And similar to how JSX lets us write HTML in with our Javascript, Styled Components let us write classic CSS, but it has many classic JS ways for us to manage our styling logic.</p>
<p>From the [documentation](<a href="https://www.styled-components.com/" title="https://www.styled-components.com/">https://www.styled-components.com/</a>): Styled components combine the html element and the rules that style it.</p>
<h2 id="installation-and-basic-use-of-styled-components"><a aria-hidden="true" tabindex="-1" href="#installation-and-basic-use-of-styled-components"><a href="#installation-and-basic-use-of-styled-components" style="margin-right: 10px">#</a></a>Installation and basic use of Styled Components</h2>
<p>Styled Component are quite easy to use. First install the module:</p>
<pre><code>yarn add styled-components
</code></pre>
<p>And here is a basic example of how to use it:</p>
<pre><code>import styled from 'styled-components';

const Wrapper = styled.div`
  height: 100%;
  width: 100%;
`;

const Header = styled.header`
  width: 100%;
  height: 70px;
  background-color: #333;
  border-bottom: 1px solid #333;
  display: flex;
`;

const Body = styled.div`
  background-color: #fff;
  color: #333;
`;

export default function App(props) {
  return (
    &#x3C;Wrapper>
      &#x3C;Header/>
      &#x3C;Body>
        &#x3C;p>Hello Styled Components&#x3C;/p>
      &#x3C;/Body>
    &#x3C;/Wrapper>
  );
}
</code></pre>
<p>As you can see, you just create a new <code>const</code> using the <code>styled</code> object we imported. That new <code>const</code> is now a component we can use when we render. So no more dealing with classNames, the connection from the component to its styling is made directly.</p>
<p>Because its CSS we can use all the classic CSS features like media queries, nesting, and psudo-selectors.</p>
<p>We can also simple interpolate, like this example from [the documentation](<a href="https://www.styled-components.com/" title="https://www.styled-components.com/">https://www.styled-components.com/</a>).</p>
<pre><code>const Button = styled.button`
  border-radius: 3px;
  padding: 0.25em 1em;
  margin: 0 1em;
  background: transparent;
  color: palevioletred;
  border: 2px solid palevioletred;

  ${props => props.primary &#x26;&#x26; css`
    background: palevioletred;
    color: white;
  `}
`;
</code></pre>
<p>When you interpolate a function in the CSS string definition like this, that function automatically has access to the props from the component it is being used in. Thats pretty neat.</p>
<p>You can also attach props to the Styled Component and then add interpolated logic based on those props. For example:</p>
<pre><code>// In the render:
&#x3C;Header size="3em">&#x3C;/Header>

// In the Styled Component definition
const Header = styled.div`
  size: ${props => props.size || '1em'}
`
</code></pre>
<p>Or if we are trying to just extend an existing styled component without any logic we can do this:</p>
<pre><code>const Button = styled.button`
  color: palevioletred;
  font-size: 1em;
  margin: 1em;
  padding: 0.25em 1em;
  border: 2px solid palevioletred;
  border-radius: 3px;
`;

// We're extending Button with some extra styles
const TomatoButton = Button.extend`
  color: tomato;
  border-color: tomato;
`;
</code></pre>
<p>You can also wrap any existing component.</p>
<pre><code>const HelloComponent = props => &#x3C;div>Hello Styled Components!&#x3C;/div>
const StyledComponent = styled(HelloComponent)`
  color: blue;
`
</code></pre>
<p>This is particularly useful when working with libraries like <code>react-router</code> because they give you components like <code>Link</code> to use.</p>
<pre><code>// Link component from react-router
const StyledLink = styled(Link)`
  color: pink;

  &#x26;:hover {
    text-decoration: none;
  }
`;
</code></pre>
<p>There are a few more tricks available. Checkout the [full documentation](<a href="https://www.styled-components.com/docs" title="https://www.styled-components.com/docs">https://www.styled-components.com/docs</a>) for more.</p>
<p>Styled Components, allow us to really simplify our display logic while still being able to customize our styling in complex ways. Oh yeah, it also works with React Native.</p>
<p>So give Styled Components a try!</p>
