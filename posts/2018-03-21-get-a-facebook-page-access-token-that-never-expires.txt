b:T105a,<p><strong>IMPORTANT! -</strong> Make sure that your user has permissions to access both the Facebook page and the Facebook groups, and make sure that the Facebook groups are linked to the Facebook page!</p>
<p>Now lets talk about some definitions:</p>
<p><strong>short-lived user access token</strong>: This is the regular user_access_token, and it typically lasts about 2 hours.</p>
<p><strong>long-lived user access token</strong>: This user access token last about 60 days and you get it by extending a short-lived user access token</p>
<p><strong>page access token</strong>: These are similar to user access tokens and are created using a user access token.</p>
<p>If you create a page access token using a short-lived user access token, it is also short-lived and expires in about 2 hours.</p>
<p>But if you create a page access token using a long-lived user access token, then it does not have an expiration date. It lasts until its access is revoked.</p>
<p>Bingo!</p>
<p>To get a user access token you first have to have a facebook app, and then the user (in this case, you) has to grant permissions to your app. Specifically your app needs the permission called <code>manage_pages</code>.</p>
<p>The easiest way to do this is to use <a href="https://developers.facebook.com/tools/explorer">Facebook's Graph API Explorer</a>.</p>
<p>Open up the page, then select your app Application dropdown and select "Get User Access Token" from the other dropdown:</p>
<p><img src="/uploads/2018/03/21/Graph_API_Explorer_-_Facebook_for_Developers-1.png" alt=""></p>
<p>Then in the popup, make sure you check the box for <code>manage_pages</code> and click "Get Access Token"</p>
<p><img src="/uploads/2018/03/21/Graph_API_Explorer_-_Facebook_for_Developers2.png" alt=""></p>
<p>Then your access token will be populated in the Access Token field:</p>
<p><img src="/uploads/2018/03/21/Graph_API_Explorer_-_Facebook_for_Developers3.png" alt=""></p>
<p>Ok, then we need to exchange that short-lived token for a long-lived token, and we can do it with this same Facebook Graph API Explorer tool.</p>
<p>We need to make a GET request to the Facebook Graph API to the path <code>/oauth/access_token</code> with the following parameters:</p>
<pre><code>/oauth/access_token?  
    grant_type=fb_exchange_token&#x26;           
    client_id={app-id}&#x26;
    client_secret={app-secret}&#x26;
    fb_exchange_token={short-lived-token-we-just-made} 
</code></pre>
<p>So make a string that looks like this:</p>
<pre><code>/oauth/access_token?grant_type=fb_exchange_token&#x26;client_id=9382767696389008&#x26;client_secret=1aac234d48ce90uu57c1b579faa92f00ufa&#x26;fb_exchange_token=EAAcZBbBDPXSwBACfeRskLz5LJJCigZBZBDyr4pmZCcVsrpFaoDUDSXthSDzdq4bqO75iEE17TBbpxIyQ1eTR3OIfGq2KPZBloF4xJ0I1GU1hxOnCGJmZBVg0DXU4IrDgcURm4ceKE4M1ZAFdGNmt5y2shsETLxAEKwZD
</code></pre>
<p>And paste it into here and you'll get your long-lived access token in the response:</p>
<p><img src="/uploads/2018/03/21/Graph_API_Explorer_-_Facebook_for_Developers4-1.png" alt=""></p>
<p>Now that you have your long-lived user access token, we can exchange it by making a GET request in the Facebook Graph API Explorer to get the long-lived page access token.</p>
<p>You need to take that long-lived user access token and paste it into the Access Token field then put this in the path field:</p>
<pre><code>{fb-page-id}]/?fields=access_token
</code></pre>
<p>Then you will get your long-lived page access token in the response!</p>
